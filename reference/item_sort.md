# Sort Items

Sort Items

## Usage

``` r
item_sort(items, data.model)
```

## Arguments

- items:

  a data.frame of the items.

- data.model:

  a data.frame of the data model.

## Value

A data.frame of the items, sorted based on the data model.

## Examples

``` r
if (FALSE) { # \dontrun{
item_sort(items, data.model)
} # }
```
