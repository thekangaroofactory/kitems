# Sort items

Sort items

## Usage

``` r
item_sort(items, data.model)
```

## Arguments

- items:

  a data.frame of the items

- data.model:

  a data.frame, defining the data model

## Value

a data.frame of the items, sorted based on the data model

## Examples

``` r
if (FALSE) { # \dontrun{
item_sort(items, data.model)
} # }
```
