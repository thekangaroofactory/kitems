# Import Module Server

The import module server is used as a child of the admin module server.

## Usage

``` r
import_server(id, k_data_model, k_items, callback)
```

## Arguments

- id:

  the id of the module.

- k_data_model:

  the reference of the data model reactive.

- k_items:

  the reference of the items reactive.

- callback:

  a reactive to call once import is finished.
