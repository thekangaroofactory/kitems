# Test date attribute

Test date attribute

## Usage

``` r
hasDate(data.model)
```

## Arguments

- data.model:

  a data.frame containing the data model

## Value

a logical, TRUE if data model has an attribute 'date' (exact match)

## Examples

``` r
if (FALSE) { # \dontrun{
hasDate(data.model = mydatamodel)
} # }
```
