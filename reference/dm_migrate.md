# Data Model Migration

Execute the migration procedure on a data model.

## Usage

``` r
dm_migrate(data.model)
```

## Arguments

- data.model:

  a data.frame of the data model.

## Value

The migrated data model or `NA` if no migration is required.

## Examples

``` r
if (FALSE) { # \dontrun{
dm_migrate(data.model)
} # }
```
