# Admin Widget

Admin Widget

## Usage

``` r
admin_widget(id)
```

## Arguments

- id:

  the id used to launch the admin module server

## Value

A
[`htmltools::tagList()`](https://rstudio.github.io/htmltools/reference/tagList.html)
object containing the admin UI

## Examples

``` r
if (FALSE) { # \dontrun{
admin_widget(id = "mydata")
} # }
```
