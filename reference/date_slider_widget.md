# Date Slider Widget

The date slider UI component to drive the date filter.

## Usage

``` r
date_slider_widget(id)
```

## Arguments

- id:

  the server module id.

## Value

An HTML component to include on UI side.

## Examples

``` r
if (FALSE) { # \dontrun{
date_slider_widget(id = "mydata")
} # }
```
