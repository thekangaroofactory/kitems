# Delete Item(s)

Delete Item(s)

## Usage

``` r
rows_delete(items, id)
```

## Arguments

- items:

  the data.frame of the items

- id:

  a vector of id(s) for the item to delete

## Value

the new data.frame of the items

## Examples

``` r
if (FALSE) { # \dontrun{
rows_delete(items = myitems, id = 123456789)
} # }
```
