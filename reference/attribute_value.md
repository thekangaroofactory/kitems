# Turn Value(s) into Attribute

Turn Value(s) into Attribute

## Usage

``` r
attribute_value(key, value, data.model)
```

## Arguments

- key:

  the name of the attribute

- value:

  a vector of input values

- data.model:

  the data.frame of the data.model

## Value

a vector, with values checked against the data.model

## Examples

``` r
if (FALSE) { # \dontrun{
attribute_value(key, value, data.model)} # }
```
