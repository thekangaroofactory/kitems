# Danger Zone Widget

Danger Zone Widget

## Usage

``` r
danger_zone_widget(k_data_model, ns)
```

## Arguments

- k_data_model:

  the reference of the data model reactive object.

- ns:

  the namespace function.

## Value

A tagList object to include in the UI.

## Examples

``` r
if (FALSE) { # \dontrun{
danger_zone_widget(k_data_model, ns)
} # }
```
