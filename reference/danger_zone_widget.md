# Danger Zone Widget

Danger Zone Widget

## Usage

``` r
danger_zone_widget(k_data_model, ns)
```

## Arguments

- k_data_model:

  the reference of the data model reactive value

- ns:

  the namespace function

## Value

a tagList object

## Examples

``` r
if (FALSE) { # \dontrun{
danger_zone_widget(k_data_model, ns)
} # }
```
