# Extract colClasses from data model

Extract colClasses from data model

## Usage

``` r
dm_colClasses(data.model = NULL)
```

## Arguments

- data.model:

  a data.frame containing the data model

## Value

a named vector of the attribute types

## Examples

``` r
if (FALSE) { # \dontrun{
dm_colClasses(data.model = mydatamodel)
} # }
```
